## Development

To start, run the following commands.

> To seed database with MongoDB:
```sh
npm run seed
```
> Start Webpack:
```sh
npm run build
```
> Start Server:
```sh
npm start
```

## Dependencies

Run the following command to install dependancies

```sh
npm install
```

This should give you all the necessary files to get started.
